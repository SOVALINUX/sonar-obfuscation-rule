This rule detects usage of parameter annotations without specifying value (i.e. parameter name). So it can lead to errors after obfuscation.

Validated annotations are @PathParam, @RequestParam, @QueryParam

Noncompliant Code Example

... endpointMethod(@PathParam String type) {...}
... endpointMethod(@RequestParam(required = false) String type) {...}

Compliant Solution

... endpointMethod(@PathParam(value = "type") String type) {...}
... endpointMethod(@RequestParam("type") String type) {...}
